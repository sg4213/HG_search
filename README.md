# HG_search

For each bp, corresponding structure coordinates and structure factor data were downloaded from PDB-REDO (https://pdb-redo.eu/). To enable a fair comparison, Watson-Crick and Hoogsteen models were refined in an identical manner using the same experimental structure factor amplitudes and refinement procedure. 
Watson-Crick model was generated by reciprocal space maximum likelihood refinement of the entire model against experimental structure factor amplitudes using phenix.refine with default geometric restraint and atomic displacement factor settings in the PHENIX software [14, 22]. 
To generate Hoogsteen model, the purine base (A/G) in the parent structure was rotated 180˚ about the glycosidic bond (C1’-N9) using in-house python script. The resulting structure was subjected to two set of refinements. First, to pre-position the rotated base, reciprocal space maximum likelihood refinement, with the positions of all the atoms except for those in the flipped nucleotide and its immediate neighboring nucleotides (±2 nucleotides in the 3’ and 5’ directions) fixed. For G-C bps, the C-N3 was protonated using pymolPy3 (http://www.pymol.org/) package prior to refinement. Next, a second round of global reciprocal space maximum likelihood refinement of the entire model in the same way as for Watson-Crick model.
